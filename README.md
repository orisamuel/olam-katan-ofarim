# עולם קטן · הירוקה בעופרים 🌿

מיניסייט הומוריסטי (אתר אחות ל־[datiim-shluvim](https://github.com/orisamuel/datiim-shluvim)).

מכירים לך **זוג השבוע** — נועם ויעל שגרים בהירוקה בעופרים — ומנהלים איתך **שיחת בילוש** קצרה כדי למצוא את החוליה שמחברת ביניכם. למערכת "יש" תיק פרטים על הזוג, אבל היא **לא מגלה אותם** — רק שואלת, ורק אם קלעת חושפת את הקישור. והקישור לא בנאלי:

- שאלות **מתגלגלות** (כרטיס אחרי כרטיס), כל אחת ממשיכה את הקודמת בניסוח מקשר.
- **שאלה ראשונה: גבר / אשה** — ומכאן השאלות מתאימות את עצמן (אצל גבר "אשתך עשתה שירות?", אצל אשה "בעלך למד בישיבה?").
- כל שאלה **סגורה וחשופה**, בלי רמז: *"למדת במרכז הרב?"*
  - **כן** → חושפת קישור מפתיע דרך צד־שלישי: *"אז אתה מכיר את יענקל׳ה מהמכולת שממול — זה הדוד של נועם, והשדכן שלהם."* 🤯
  - **לא** → *"אין בעיה, אז איפה כן?"* (נפתחת לשאלה פתוחה) → ממשיכים לכיוון הבא, **בלי לגלות כלום**.
- הכיוונים: ישיבה → שירות לאומי → קרוב משפחה ביישוב → שבט בבני עקיבא. הראשון שקולע = החוליה. אם אף אחד לא קלע — נסגר בחיבור דרגה־שנייה ("עולם קטן, מכר אחד ביניכם").

## החלפת זוג שבועית
כל הזוג מוגדר באובייקט `COUPLE` בראש ה־`<script>` שב-[index.html](index.html). כדי לשים זוג חדש בכל שבוע — עורכים רק את האובייקט הזה:
- `names` / `bio` — שם ותיאור הזוג.
- `avenues[]` — לכל כיוון: `probe` (השאלה הסגורה), `yes` (תווית כפתור "כן"), `hit` + `revealLine` (הקישור המפתיע אם קלעת), `openQ` + `openChips` (אם פספסת), ו-`weave` (חיבור דרגה־שנייה).
- שדות שהם `{m, f}` מתאימים את עצמם אוטומטית למין שנבחר. `spouseFor` מסמן לאיזה מין הכיוון מדבר על בן/בת הזוג (מוסיף אופציית "עוד לא נשוי/אה" + `singleReact`).

אין צורך לגעת בשאר הקוד.
