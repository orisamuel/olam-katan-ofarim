# עולם קטן · הירוקה בעופרים 🌿

מיניסייט הומוריסטי (אתר אחות ל־[datiim-shluvim](https://github.com/orisamuel/datiim-shluvim)).

הפעם מזווית אחרת: מכירים לך זוג חמוד — **נעם ויעל + 2 ילדים** שגרים בהירוקה בעופרים — ובודקים דרך שרשרת שאלות (איפה למדת בתיכון, ישיבה/מכינה, שירות לאומי, קרוב משפחה ביישוב, שבט בבנ״ע) **דרך כמה אנשים אתה כבר קשור אליהם**.

הפאנץ' בסוף: *"יעל עשתה שירות עם בת הדודה שלך! עולם קטן, אה? יאללה בוא לגור בעופרים."*

## הרצה
דף סטטי בודד. פותחים את `index.html` בדפדפן, או דרך GitHub Pages.

- טכנולוגיה: HTML/CSS/JS וניל, ללא תלויות, ללא שרת. כל הנתונים נשארים בדפדפן.
- עברית RTL, פונטים Rubik/Heebo.
