# עולם קטן · הירוקה בעופרים 🌿

מיניסייט הומוריסטי (אתר אחות ל־[datiim-shluvim](https://github.com/orisamuel/datiim-shluvim)).

מכירים לך **זוג השבוע** — נועם ויעל שגרים בהירוקה בעופרים — ומנהלים איתך **שיחת בילוש** קצרה כדי למצוא את החוליה שמחברת ביניכם. למערכת "יש" כמה פרטים אמיתיים על הזוג, והיא מנסה כיוון אחרי כיוון:

- שאלה **סגורה** תלוית פרט אמיתי: *"למדת במרכז הרב?"*
  - **כן** → מצאנו! נועם גר בקריית משה לידה, בטוח נתקלתם. 🎯
  - **לא** → *"אוקיי, אז איפה כן?"* (נפתחת לשאלה פתוחה) → ועוברים לכיוון הבא: *"אולי דרך אשתך?"*
- ממשיכים דרך ישיבה → שירות לאומי של יעל → קרוב משפחה ביישוב → שבט בבני עקיבא, עד שמוצאים חוליה. אם אין חיבור ישיר — נסגר בחיבור דרגה שנייה ("עולם קטן, מכר אחד ביניכם").

## החלפת זוג שבועית
כל הזוג מוגדר באובייקט `COUPLE` בראש ה־`<script>` שב-[index.html](index.html). כדי לשים זוג חדש בכל שבוע — עורכים רק את האובייקט הזה:
- `names` / `bio` / `sub` — שם ותיאור הזוג.
- `avenues[]` — לכל כיוון: `probe` (השאלה הסגורה עם הפרט הידוע), `hit` + `revealLine` (תגובה אם קלעת), `openQ` + `openChips` + `openReact` (אם פספסת), ו-`weave` (ניסוח לחיבור דרגה־שנייה).

אין צורך לגעת בשאר הקוד.
